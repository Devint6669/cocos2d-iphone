Creating a new Target

Duplicate an existing Target

  1. In "Groups & Files" expand "Targets".
  2. Select one of the demos, menu/right click on it and choose "Duplicate"
  3. Rename the project  "MySuperApp"

Modify your new Target

  1. With "MySuperApp" selected, menu click it and "Get Info"
  2. In "Build" make sure the "Product" is "MySuperApp"
  3. In "Properties" make sure "Identifier" has your URL which is registered with Apple and is part of your device's provision (eg. com.yourcompany.*)
  4. Close the "Info" window

Remove unwanted files from your new Target

  1. With the new target selected look at the list of files in the top of the split pane
  2. Now find the .m files that used to be part of the target you duplicated (eg. if you've duplicated "MenuTest", find and select  "MenuTest.m") then remove it by deleting it from the target
  3. Do the same for any resources that were used in "MenuTest" that you don't need in "MySuperApp"

Add new files to your new Target

  1. With "MySuperApp" target selected, expand it to see it's three folders: "Copy Bundle Resources", "Compile Resources" and "Link Binary with Libraries"
  2. Add my new .m file "MySuperApp.m" and make sure it is part of the new target in its "Compile Sources"
  3. I also add any new resources specific to MySuperApp" by dragging them from "Groups & Files/cocos2d-port/Resources" to "Copy Bundle Resources"
Check everything builds and you are ready!
