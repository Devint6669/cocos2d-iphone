TileMapAtlas tools

 

pgu's tile editor: pgu_leveleditor-20081117.tar.gz

This is the 'official' tool to generate map files (.tga files). If you are using cocos2d 0.5.2 or earlier you need to convert your .tga file to NON RLE image. v0.5.3 works OK with RLE TGA images.

To generate tiles files, you can use this tool, or any other imaging program. The 'tiles.tga' file  shall be converted to 'tiles.png' file to be used inside cocos2d.


You will need python, pygame and pyobjb-c to run this tool:

  * pygame 1.8.1 for OSX 10.5

  * pyobjc-1.4-py2.5-macosx10.4.mpkg.zip

  * If the pygame installation fails, try installing Mac Python first: python-2.5-macosx.dmg


Read this article for further info: How to use Tile Map editor for cocos2d

 

Ernesto Corvi's tile generator: tiler.zip

Tiler is a small command line utility that tiles an arbitrary image
into a PNG texture containing tiles and a TGA image containing a tilemap.
You can specify the tile size (must be a power of 2) with the -tilesize option.
